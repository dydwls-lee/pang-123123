# 메인 화면 (초안)

`docs/PRD.md` 기준 팡(Pang) 게임의 첫 진입 화면 구성.

## 목적

플레이어가 게임에 진입했을 때 가장 먼저 보는 화면으로, 게임 시작과 기본 정보 확인을 담당한다.

## 구성 요소

- **타이틀 로고**: 게임 이름(팡) 표시
- **시작 버튼**: 클릭/입력 시 Mission 1으로 진입
- **조작 안내**: 좌/우 이동, 와이어 발사 키 안내 (예: ←/→, Space)
- **최고 점수(하이스코어)**: 이전 플레이 최고 점수 표시 (로컬 저장 기준)
- **버전/크레딧 정보** (선택): 하단 작게 표시

## 화면 흐름

1. 앱 진입 → 메인 화면 렌더링
2. 시작 버튼(또는 지정 키) 입력 → Mission 1 게임 화면으로 전환
3. 게임 오버 후 메인 화면으로 복귀 가능 (재시작 동선)

## 비고

- 초안 단계이므로 상세 레이아웃/스타일은 구현 시 확정.
- 멀티 미션 확장 시 메인 화면에 미션 선택 UI가 추가될 수 있음(현재 범위 밖).
