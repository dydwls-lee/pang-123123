# 게임 룰 상세 (초안)

`docs/PRD.md`의 핵심 목표를 기반으로 한 상세 규칙 정리.

## 플레이어

- 화면 하단에서 좌/우로 이동 가능.
- 위쪽 고정 방향으로 와이어를 발사해 공을 제거.
- 조작: 좌/우 이동 + 발사, 두 가지 입력만 사용.

## 체력(HP)

- 시작 HP는 100.
- 공에 닿을 때마다 HP 10 감소.
- HP가 0이 되면 게임 오버.

## 공

- 벽과 바닥에 튕기며 포물선으로 이동하는 물리 기반 오브젝트.
- 크기는 1단계로 고정(분열 없음).
- 와이어에 맞으면 즉시 제거(사라짐).

## 스테이지 클리어 조건

- 화면에 등장한 모든 공을 제거하면 스테이지 클리어.
- 제한 시간 내에 클리어하지 못하면 실패 처리.

## 점수

- 공 제거 시 점수 획득(세부 배점은 미션별 문서에서 정의).
- 클리어 시 남은 시간에 비례한 시간 보너스 점수 지급.

## 게임 종료 조건

- HP 0 도달 시 게임 오버.
- 제한 시간 초과 시 스테이지 실패.

## 비고

- 현재는 1인 플레이 기준으로 정의.
- 특수 무기, 지형 기믹 등은 후속 미션에서 추가 예정이며 본 문서 범위 밖.
